# 🚀 AirAware – Real-Time Air Quality Monitoring Web App

## 📌 Project Description
AirAware is a web application that helps users check **real-time air quality data** of different cities. The main goal of this project is to spread awareness about air pollution and help users take necessary precautions for their health.

The application presents air quality information in a simple and understandable way so that even non-technical users can easily interpret it.

---

## 🎯 Features
- 🌍 Search air quality by city  
- 📊 Real-time AQI (Air Quality Index) data  
- ⚠️ Health recommendations based on AQI levels  
- 🔐 User authentication (Login & Register system)  
- 📱 Responsive design for mobile and desktop
